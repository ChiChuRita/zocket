title: Rooms

Rooms are a way to group clients. Clients can subscribe to topics, join chat rooms, etc.

### Joining a Room
Use `ctx.rooms.join(roomId)` in a message handler.

```typescript
.incoming(({ send }) => ({
  joinRoom: zo.message
    .input(z.object({ roomId: z.string() }))
    .handle(({ ctx, input }) => {
      ctx.rooms.join(input.roomId);
    })
}))
```

### Leaving a Room
Use `ctx.rooms.leave(roomId)`.

```typescript
ctx.rooms.leave("room-123");
```

### Checking Membership
Use `ctx.rooms.has(roomId)`.

```typescript
if (ctx.rooms.has("admin-room")) {
  // ...
}
```
