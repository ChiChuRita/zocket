### RPC Errors
If your server handler throws an error, it is propagated to the client's promise rejection.

```typescript
// Server
.handle(() => {
  throw new Error("Validation failed");
})

// Client
try {
  await client.doSomething();
} catch (e) {
  console.error(e.message); // "Validation failed"
}
```

### Connection Errors
Listen to global connection errors using the `onError` hook or `lastError` property.

```typescript
client.onError((err) => {
  console.error("Socket error:", err);
});
```
